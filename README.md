实验二 小程序网络API：天气预报   
要求：   
（一）基本功能   
1.导航栏显示”今日天气“   
2.点击地区文字可以修改地区   
3.显示该地区的天气和温度   
4.显示天气对应的图片   
（二）额外功能（本实验额外实现）   
1.横向显示给该地区接下来24小时的天气情况。   
2.添加背景面板   
3.如果地区为雨天，增加雨点效果
